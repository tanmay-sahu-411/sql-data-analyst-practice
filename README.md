# 📊 SQL Data Analyst Practice

This repository contains structured SQL practice focused on becoming interview-ready for Data Analyst roles.

The objective is to master SQL progressively — from fundamentals to advanced analytical querying — using SQL Server (SSMS).

---

---

## 🟢 Beginner Level (Foundation)

Covers core SQL syntax and fundamental query writing:

- SELECT
- WHERE
- ORDER BY
- TOP (SQL Server)
- DISTINCT
- IN / BETWEEN / LIKE
- NULL Handling
- Aggregations (COUNT, SUM, AVG, MIN, MAX)
- GROUP BY
- HAVING
- CASE Statements
- Basic INNER & LEFT JOIN

---

## 🟡 Intermediate Level (Business SQL)

Focuses on analytical thinking and multi-table logic:

- Revenue calculations
- Contribution %
- Customer segmentation
- Multi-table JOINs
- Subqueries
- Common Table Expressions (CTE)

---

## 🔴 Advanced Level (Analyst Strength)

Advanced SQL used in real-world analytics:

- ROW_NUMBER()
- RANK() / DENSE_RANK()
- SUM() OVER()
- Running totals
- LAG() / LEAD()
- Month-over-Month growth
- Time series analysis

---

## 🎯 Goal

To build strong SQL fundamentals and analytical capability required for Data Analyst and Business Intelligence roles.
